# HTML Snapshot Viewer (Chrome Extension)

## Что делает
- Захватывает HTML текущего DOM активной вкладки (после динамических изменений на странице).
- Открывает отдельное окно просмотра.
- Позволяет Copy / Save .html.
- Есть режим “Запись действий”: на странице появляется виджет, вы раскрываете нужные элементы и нажимаете “Захватить”.

## Установка (Developer mode)
1. Откройте `chrome://extensions/`
2. Включите `Developer mode`
3. Нажмите `Load unpacked`
4. Выберите папку расширения

## Использование
- Обычный режим: откройте нужные меню/списки → нажмите “Захватить и открыть окно”.
- Режим записи: выберите “Запись действий” → “Запустить виджет на странице” → раскройте элементы → “Захватить” в виджете.

## Ограничения
- Cross-origin iframe недоступны (политика безопасности браузера).
- Shadow DOM доступен только если он **open**.
